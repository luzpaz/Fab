4 Class ModFabMount:

An operations plane that can be oriented for subsequent machine operations. This class basically corresponds to a FreeCad Datum Plane. It is basically the surface to which the 2D ModFabGeometry's are mapped onto prior to performing each operation. This class is immutable (i.e. frozen.)

Attributes:

  • Name: (str): The name of the ModFabPlane.
  • Contact (Vector): A point on the plane.
  • Normal (Vector): A normal to the plane
  • Orient (Vector): A vector in the plane that specifies the north direction when mounted in a machining vice.
  • Operations (Tuple[ModFabOperation, ...]): The operations to perform.
